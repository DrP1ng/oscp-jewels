# imap :143 :993

## nc method

```bash
nc -vn IP 143

# login
a LOGIN USER PASS
>> a OK LOGIN Ok.

# list all mailboxes
a LIST "" "*"

# see inside mailbox
a EXAMINE "<inbox-name>"
a SELECT "<inbox-name>"

# if EXISTS, we can FETCH
a FETCH <mail-number> BODY.PEEK[]
a FETCH <mail-number> BODY[]
```
