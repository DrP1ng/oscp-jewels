# dosbox

## information

```bash
/usr/bin/dosbox -version
# opens a dosbox emulator on GUI.
# need UI access to interact with it.

# manpage
https://linux.die.net/man/1/dosbox
```

```
Z:\> MOUNT <DRIVE> <PATH-TO-MOUNT>
# mount k /
# can access root-fs now at drive K:
```
